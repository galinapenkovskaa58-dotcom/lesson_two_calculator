# Калькулятор семейного бюджета

Учебный мини‑проект на чистом **HTML/CSS/JavaScript** для учёта доходов и расходов семьи.

## Что внутри

- Страница-калькулятор: `Клькулятор.html`
- Приветственный лендинг: `index.html`
- Переключение темы (светлая/тёмная) кнопкой в правом нижнем углу
- Запоминание выбранной темы через `localStorage`

## Как запустить

1. Откройте `index.html` в браузере.
2. Нажмите кнопку **«Открыть калькулятор»**.

Подсказка: для учебного проекта можно открывать файлы напрямую без сервера.

## Деплой (по желанию)

Если будете публиковать на **GitHub Pages**, убедитесь, что стартовой страницей является `index.html`.
